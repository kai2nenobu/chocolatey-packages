# May Authentication Proxies Explode

This package will download and install [busybox-w32](https://frippery.org/busybox/).

## Install Information

busybox executable is located at `C:\ProgramData\chocolatey\lib\busybox-w32\buxybox.exe` and is included in `PATH` environment variable.

On 32bit system, this package install a 32 bit executable, and on 64bit system, install a 64bit one. If you want to install 32bit executable on 64bit system, use `choco install --x86`.
